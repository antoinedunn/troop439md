# troop439md

Temporary website for Troop 439 (Kensington, Maryland), served via GitHub Pages
at [troop439md.org](https://troop439md.org).

Troop 439 is supported by Kensington Scout Sponsors, a 501(c)(3) nonprofit
organization based in Garrett Park, Maryland. The established troop website
remains available at <https://kensington439.mytroop.us/>.
